Interactive Map of World Landmarks

This interactive learning experience invites users to explore five world‑famous landmarks — from the Eiffel Tower to the Pyramids of Giza — through a clickable, visually engaging map interface. Each landmark opens a pop‑up layer featuring an image, a concise historical description, and an engaging “fun fact” to spark curiosity.

The design draws on Mayer’s Multimedia Learning Principles by pairing concise on‑screen text with relevant imagery to reduce cognitive load and increase retention. The interactive map format also supports constructivist learning, allowing learners to explore content in a self‑directed, non‑linear path.

Built in Articulate Storyline 360, this project demonstrates:

  • Custom navigation with clickable hotspots and visited state feedback
  • Multi‑layer slide design with consistent template‑driven layout
  • Light visual animation for smooth transitions
  • Integration of high‑quality, royalty‑free imagery and vector icons
  • Variable‑based tracking of visited locations (optional in expanded versions)

This project can stand alone as a microlearning activity or be embedded as an interactive element within a larger eLearning course.
