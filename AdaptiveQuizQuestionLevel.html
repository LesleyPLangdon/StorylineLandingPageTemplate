Adaptive Quiz — Question‑Level Model

This adaptive assessment evaluates the depth of knowledge for each individual concept by adjusting question difficulty immediately after each response. Each cycle starts with a hard question targeting higher‑order thinking skills such as create or evaluate:

  • Correct → move to another hard question on the next concept
  • Incorrect → progress to a medium‑difficulty question on the same concept, designed to assess skills like analyze or apply
  • Incorrect on medium → move to an easy question on the same concept, checking understanding or remembering, before starting the next cycle at hard

This approach allows the quiz to pinpoint not just if a learner knows a concept, but at what cognitive depth they can demonstrate mastery — making it especially useful for skill‑based or performance‑based assessments.

The design aligns with Bloom’s Taxonomy and principles of formative assessment, ensuring that each learner’s experience is tailored to their demonstrated mastery level for each concept. This targeted adaptation reduces frustration for learners who need foundational support while challenging those ready for advanced thinking.

Built in Articulate Storyline 360, this project demonstrates:

  • Question‑by‑question branching logic using triggers and conditional variables
  • Targeted difficulty scaling per concept based on Bloom’s cognitive levels
  • State‑based control to ensure each question is only counted once
  • Variable tracking for total correct answers, correct answers per cycle, and performance by difficulty level
  • Consistent slide templates for rapid question development
  • Chromeless player setup for distraction‑free assessment delivery

This model works well for scenarios where evaluating mastery depth per concept is important, such as skills assessments, competency testing, and targeted review activities.
