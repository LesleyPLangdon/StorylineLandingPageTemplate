Pirate Treasure Interaction

My very first Articulate Storyline 360 project — a lighthearted, pirate-themed one-question quiz inspired by a Devlin Peck tutorial. When the learner answers correctly, they’re rewarded with a key they can drag over a treasure chest to unlock it.

While simple, this interaction holds sentimental value as it marked the beginning of my journey into interactive learning design. It demonstrates custom drag-and-drop functionality, basic triggers, states, and visual feedback — all wrapped in a fun, thematic experience.
